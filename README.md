# Reliquary

<p align="center">
  <img src="assets/reliquary-logo.png" alt="Reliquary logo" width="160">
</p>

**Reliquary** is a clean, lightweight suite of creative tools made for video editors, content creators, and post-production workflows.

The goal is simple: provide small, focused apps that help prepare, inspect, repair, and optimize media files before editing, without turning the workflow into a technical mess.

Reliquary is not meant to replace a full NLE, a DAW, or professional post-production software.  
It is a toolbox for the annoying steps around editing: containers, framerates, downloads, and media preparation.

---

## About this repository

This repository is the **public release hub** for Reliquary.

It is used for:

- downloading Reliquary apps;
- tracking public issues;
- giving users a central place to access the suite.

The source code is not distributed from this repository.

---

## Available apps

### Obelisk

**License and access manager**

Obelisk handles activation, licenses, app access, and account-related features for the Reliquary ecosystem.  
It acts as the central gateway for the suite.

---

### Daedalus

**Media downloader**

Daedalus is designed to retrieve media from supported sources for legitimate creative workflows, such as personal archives, creator-owned content, authorized downloads, and production references.

Reliquary does not encourage copyright infringement or unauthorized downloading. Users are responsible for respecting platform terms, copyright law, and usage rights.

---

### Talos

**Remuxer**

Talos changes or repairs media containers without unnecessary re-encoding when possible.  
It is useful for making problematic files easier to import into editing software.

Typical use cases:

- remuxing exotic containers;
- fixing files that are awkward for editing software;
- preparing media for DaVinci Resolve, Premiere Pro, or other NLEs;
- cleaning up container-level issues.

---

### Chronos

**VFR to CFR converter**

Chronos fixes variable frame rate media by converting it to a constant frame rate workflow.

This is especially useful for:

- phone videos;
- screen recordings;
- OBS captures;
- Discord or browser recordings;
- files that drift out of sync in editing software.

## Design direction

Reliquary uses a clean, light, artifact-inspired visual identity.

The interface direction is based on:

- warm off-white backgrounds;
- charcoal text and iconography;
- antique gold accents;
- flat-design app icons;
- one signature color per app.

The goal is to feel clean and modern without losing the mythological and artifact-based identity of the suite.

## Support and feedback

When reporting a media-related issue, please include:

- the app name;
- your operating system;
- the app version;
- a clear description of the issue;
- the file format involved;
- logs or screenshots if available.

Do not upload copyrighted media files unless you have the right to share them.
